Задача

4. Створити гілку на основі майстер гілки. Внести зміни в нову гілку та завантажити їх у віддалений репозиторій. Злити зміни нової гілки в гілку master.

Розв'язання

F:\RepoTest1>git worktree list

F:/RepoTest1 2b8ae82 [main]

F:\RepoTest1>git worktree add newtree

Preparing worktree (new branch 'newtree')

HEAD is now at 2b8ae82 Create README.md

F:\RepoTest1>git log

Вношу зміни в нову гілку

cd F:\RepoTest1\newtree

echo "New line" >> file1.txt

echo New line >> file2.txt

git status

(use "git add ..." to update what will be committed)

git add .

git commit -m "Modify file1.txt, file2.txt in worktree newtree"
